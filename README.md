# 🍽️ 餐厅扫码订餐系统 - 前端

基于 Vue 3 + Vite 的多端餐厅点餐系统前端。

## 📦 项目结构

```
├── admin/          # 商家管理后台
├── super-admin/    # 超级管理后台
├── kitchen/        # 厨房端
├── miniapp/        # 小程序端（H5）
└── docker-compose.yml
```

## 🚀 快速开始

```bash
# 进入对应目录
cd admin  # 或 super-admin / kitchen / miniapp

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🛠️ 技术栈

- **框架**: Vue 3
- **构建工具**: Vite
- **包管理**: npm
